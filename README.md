# 🚀 Rota Dev

Um planner de estudos para desenvolvedores iniciantes.

## 💡 Sobre o projeto
O Rota Dev ajuda pessoas que estão começando na programação a criarem um plano de estudos personalizado baseado em seus objetivos.

## 🛠️ Tecnologias
- React
- TypeScript
- Vite
